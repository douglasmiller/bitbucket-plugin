Destination branch to trigger the job for. If left blank, the '**' wildcard will be used to match all branches.

Supported options:

  • Comma separated list
    Multiple branches can be specified with a comma separated list.
    E.g. develop, master
  • Wildcards
    The '*' wildcard matches all characters except for '/'. The '**' wildcard matches all characters.
    E.g. release/*, mast*
